🌐 Browser Agent

AI-powered headless Chrome that completes web tasks on your behalf

How It Works

  • Request any web task via voice call or chat
  • Task is queued and runs asynchronously in headless Chrome (Playwright)
  • AI agent navigates, clicks, fills forms, reads content, and takes screenshots
  • Results are emailed to you automatically when complete
  • If the agent needs more info, it emails you a question — just reply to continue

Capabilities

Reservations Book restaurants via OpenTable, Resy, or restaurant websites
Research Look up property details on Zillow, Redfin, Realtor.com
Shopping Price comparison and product research across websites
Forms Fill out web forms, applications, and submissions
Monitoring Check website status, content changes, availability
General Any web task — if a human can do it in Chrome, the agent can try

Browser Tools (Internal)

  • navigate — Go to any URL
  • click — Click elements by CSS selector, text, or role
  • fill — Type text into form fields
  • select_option — Choose from dropdowns
  • press_key — Press keyboard keys (Enter, Tab, Escape, etc.)
  • scroll — Scroll page up or down
  • screenshot — Capture current page state
  • read_page_text — Extract visible text from the page
  • search_google — Quick Google search shortcut
  • wait — Pause for dynamic content to load
  • done — Signal task completion with summary
  • need_info — Pause and email the user a question

Safety Rules

  • Never enters payment or credit card information
  • Never enters passwords or login credentials
  • Never enters SSN, bank accounts, or sensitive financial data
  • Escalates CAPTCHAs to the user

📊 CMA Engine

Quick Property Lookup

  • Instant property snapshot from just an address
  • Returns: tax data, MLS listing, nearby sold comps, demographics, flood risk

MethodEndpointDescription
