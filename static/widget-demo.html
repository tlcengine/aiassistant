CertiHomes AI Widget

An embeddable chat widget that adds AI-powered real estate assistance to any website with a single line of code.

How to Install

Add the CertiHomes AI Assistant to any website by including one script tag before the closing </body> tag:

Add this to any page
<script src="https://aiassistant.certihomes.com/widget.js"></script>

That's it! The widget will automatically appear as a floating chat bubble in the bottom-right corner of your page. Click it to try it out now.

Features

Zero Config

Just add the script tag. No API keys, no setup, no dependencies needed.

Shadow DOM

Completely isolated CSS via Shadow DOM. No conflicts with your existing styles.

Mobile Responsive

Automatically adapts to full-width on screens under 768px.

Dark Theme

Sleek dark interface that looks great on any website design.

Real-Time AI

Powered by Claude, with access to MLS data, market stats, and property info.

Markdown Support

AI responses render bold, italic, and links for rich formatting.

Optional Configuration

You can customize the widget position and accent color using data attributes:

Custom position and color
<script src="https://aiassistant.certihomes.com/widget.js" data-position="bottom-left" data-color="#38bdf8"> </script>
